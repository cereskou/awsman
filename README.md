# awsman
AWS Assistant written in go

## in development

## Run as a web server
support rest api. please check section swagger.

## Features
1. Cognito
1. CodeCommit
1. CloudTrail
1. ELB

## Tech
1. User login (JWT+Access token bearer)
1. Swagger (swag auto-generate)
1. Web Framework echo/v4
1. Database SQLite
1. gorm
1. AWS SDK
